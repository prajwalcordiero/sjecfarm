# **App Name**: Farm Fresh Finds

## Core Features:

- Image Upload: Authorized users can upload images of products with names and descriptions using a secure form.
- Image Display: Display product images, names, and descriptions in a visually appealing grid on the client-side website.
- Image Deletion: Authorized users can delete outdated product images from the server using a secure interface.
- User Authentication: Implement a hardcoded login system to differentiate between client-side users and authorized administrators.

## Style Guidelines:

- Primary color: Earthy green (#8FBC8F) to reflect the natural, fresh produce.
- Background color: Off-white (#F5F5DC) to provide a clean and inviting backdrop.
- Accent color: Warm brown (#A0522D) to highlight calls to action and important information. 
- Body and headline font: 'PT Sans', a humanist sans-serif, combining modern with approachable feel, for both headlines and body text.
- Use clean, minimalist icons representing categories of produce like bread, vegetables, and eggs.
- Employ a grid-based layout to showcase products effectively, ensuring responsiveness across devices.
- Incorporate subtle transitions when loading images or interacting with product listings.
